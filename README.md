# Data-Science-for-Finance
Application of five trading strategies on the Sp500 sectors : 
1. Bollinger bands
2. Moving Averages
3. RSI
4. MACD
5. ADX

On three date ranges:
1. 1.1.2010-now
2. 1.1.2019-now
3. 1.1.2022-now
